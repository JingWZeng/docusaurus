> 一次http请求至少进行两次的数据转换,发送请求的时候一次,获取响应数据的时候一次

| 顺序 | 层名   | 格式                             |
| ---- | ------ | -------------------------------- |
| 1    | 应用层 | 数据                             |
| 2    | 传输层 | 数据+源端口+目标端口             |
| 3    | 网络层 | 数据+源端口+目标端口+源IP+目标IP |
| 4    | 链路层 | 转换成数字信息进行传输           |

#### 从客户端 1->2->3->4

#### 从服务端 4->3->2->1
